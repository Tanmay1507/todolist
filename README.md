A streamlined, high-performance task management application. This project focuses on clean UI/UX principles and efficient state management to help users organize their daily workflow without clutter.

---

## üöÄ Key Features

* **Real-time Interaction:** Add, edit, and remove tasks with zero latency.
* **State Persistence:** Integrated with `LocalStorage` (or Database) to ensure tasks remain saved after browser refreshes.
* **Dynamic Filtering:** Filter tasks by "All", "Active", and "Completed" status.
* **Responsive Engine:** Fully optimized for Mobile, Tablet, and Desktop views using CSS Flexbox/Grid.
* **Keyboard Accessibility:** Full support for `Enter` to add and `Tab` navigation for accessibility compliance.
